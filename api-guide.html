Hướng dẫn lấy Groq API Key

  1. Truy cập https://console.groq.com/keys
  2. Đăng nhập / Đăng ký tài khoản (Google hoặc Github).
  3. Bấm Create API Key, đặt tên tùy ý và copy chuỗi Key (bắt đầu bằng gsk_...).
  4. Dán Key vào ô của extension và bấm Bắt đầu Dịch.

Groq API có gói miễn phí (Free tier) với giới hạn request nhất định, đủ để dùng cá nhân dịch video YouTube. Không phải trả phí.
